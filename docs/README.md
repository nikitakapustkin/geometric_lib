# Лабораторная работа №2
Библиотека для нахождения периметра и площади различных геометрических фигур. Все функции написаны на языке программирования Python.
## Площадь
- Круг: S = πR²
```
R = 13
area(R) = 530.929158456675
```
- Прямоугольник: S = a*b
```
a = 12
b = 10
area(a, b) = 120
```
- Квадрат: S = a²
```
a = 12
area(a) = 144
```
- Треугольник: S = a*h/2
```
a = 10
b = 6
area(a, b) = 30
```
## Периметр
- Круг: P = 2πR
```
R = 9
perimeter(R) = 56.548667764616276
```
- Прямоугольник: P = (a + b) * 2
```
a = 8
b = 7
perimeter(a, b) = 30
```
- Квадрат: P = 4*a
```
a = 25
perimeter(a) = 100
```
- Треугольник: P = a + b + c
```
a = 3
b = 4
c = 5
perimeter(a, b, c) = 12
```
